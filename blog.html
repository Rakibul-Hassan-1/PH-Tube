Var: Var is globally scoped.Var can be updated & re-declared.
let: Var is block scoped.let can be updated but not re-declared.
const: Var is block scoped.let can neither be updated nor be re-declared.

Null:Null in javascript means an empty value and is also primitive type.
Undefined:Undefined means the variable has been declared but its value has not been assigned.

REST API: Rest Api is a service which is built on the REST architecture is called a RESTful service. It enables web applications that are built on various programming languages to communicate with each other.
